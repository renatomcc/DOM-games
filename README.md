# DOM Games
A few games created with HTML, CSS and Javascript with DOM manipulation.

## Link
You can check the project [here](https://domgames-renatomcc.netlify.app)

## Games
* Tic Tac Toe
* 2048
* Flappy Bird
* Rock Paper Scissors
* Hangman
* Wordle
* Memory
* Genius
